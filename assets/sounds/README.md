# Sound Effects for Slot Machine

This directory contains sound effects for the Chaos Slots game.

## Required Sound Files:

1. **spin.mp3** - Reel spinning sound (plays when spin button is pressed)
2. **win.mp3** - Regular win sound (plays when you win a small amount)
3. **big-win.mp3** - Big win sound (plays when you win 5x+ your bet)
4. **button.mp3** - Button click sound (plays for bet/lines adjustments)
5. **bonus.mp3** - Bonus activation sound (plays when free spins start)

## Adding Sound Files:

1. Place your sound files in this directory
2. Make sure they are in MP3 format
3. Recommended duration: 1-3 seconds each
4. Keep file sizes small for better performance

## Temporary Setup:

If you don't have sound files yet, the app will still work but won't play sounds.
You can find free slot machine sounds on sites like:
- Freesound.org
- Zapsplat
- Mixkit

## Volume Settings:

Sounds are played at 50% volume by default but can be adjusted in the code.
